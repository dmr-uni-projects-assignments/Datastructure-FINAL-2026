## Running

1. Download [Temurin JDK 17](https://adoptium.net/temurin/releases?version=17&os=any&arch=any) for your platform
2. Clone the repo: `git clone https://github.com/dmr-uni-projects-assignments/Datastructure-FINAL-2026/`
3. Enter the directory: `cd Datastructure-FINAL-2026`
4. Run the game: `gradlew run` or `.\gradlew run`
